**Course:** ICT602 

## Description
This mobile application helps investors calculate their Unit Trust dividends. It calculates both monthly and total dividends based on the invested fund, annual rate, and duration.

## Features
- **Input Validation:** Ensures months do not exceed 12.
- **Accurate Calculation:** Uses standard formulas with 2 decimal formatting.
- **Responsive UI:** Clean interface with Action Bar and Navigation Menu.
- **Reset Function:** Clears all inputs easily.
